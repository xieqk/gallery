{"posts":[{"title":"2020 国庆（10-03~10-08）","content":"重庆→武隆(天坑地缝)→成都(杜甫草堂、成都博物馆、春熙路太古里、宽窄巷子)→卧龙(自然博物馆)→猫鼻梁→双桥沟→日隆镇→长坪沟→成都(建设巷、东郊记忆)→成都(大熊猫繁育研究基地) Day1：重庆（解放碑、洪崖洞） 一早坐动车前往重庆，下午到达后在酒店附近（重庆北站附近）吃了一顿火锅（狮子坪忘年椒老火锅，味道还行）。后乘地铁前往市中心，先逛了解放碑，后前往洪崖洞。由于交通管制，去洪崖洞是绕了很长的路并拍了很久的队。虽然进入了洪崖洞，但到晚10点后，洪崖洞关灯，并没有看到洪崖洞的全貌。出去后，步行穿越千厮门嘉陵江大桥，后乘地铁返回酒店。 Day2：重庆武隆（天坑地缝） 清晨，司机从酒店将我们接走后，从高速前往武隆天坑地缝国家地质公园。到乌江边后，为赏乌江江景，便下高速沿着319国道（乌江边）继续前行。途中在乌江赤壁观景台休整并游览了乌江画廊。后继续沿着乌江前往景区。 到达天坑后，徒步下坑游览（其实买的套票带电梯票可直达坑底，没注意），坑底景色很壮观，为喀斯特地貌。沿着岩壁顺着暗河前行，感受大自然的鬼斧神工。游览约3个小时后到达前往地缝的乘车点，乘电瓶车前往地缝（也可步行）。 地缝泉水流瀑挂壁，险峻幽深，藤萝盘绕。其中一段，飞瀑咆哮，正值丰水期，即使身穿雨衣，头打雨伞，全身上下也被飞溅扑了个彻彻底底，过了便是一线天。地缝不到2小时可游览结束，但电梯排队上坑却花了一个小时左右。上坑后，驾车返回重庆住宿。 Day3：成都（杜甫草堂-成都博物馆-春熙路太古里-宽窄巷子） 第三天一大早，乘火车前往成都（第一次坐软卧，不过3小时就到了）。到火车北站后，乘公交前往酒店。稍作休整后，继续乘公交前往今天的第一个景点——杜甫草堂。草堂里茂林修竹，十分舒服，不知道杜甫能不能想得到千年后会有这么多人重新来到这里参观其曾经住过的茅草屋。草堂完整保留着清代嘉庆年间重建时的格局，里面还发掘出了唐代生活的遗址和一些唐代文物，约2小时可游览结束。 后乘车到了天府广场，并前往成都博物馆进行参观。馆1楼为希望美术特展，排队较多。由于对西方画作不是那么感兴趣，便直接前往了2楼。2~3楼为成都历史文化陈列（古代），馆藏着几百件精品文物从先秦时期的古蜀到近代西南大都会表现了成都历史的瑰丽神奇。4楼为近代陈列及当地民俗展示，讲述了城市文明的发展历程。5楼为皮影展及木偶展，展示了皮影及木偶戏的历史和现在。到顶层简单休息观光后，乘电梯前往负1楼。负1楼为人与自然的捐赠站，展出了300多件动物标本，展示了多种物种及生态系统，阐述人与自然之间的关系，特别适合小孩子游玩。 出来后，前往文殊院附近吃了严太婆锅盔（牛肉锅盔可），以及张老二凉粉（甜水面可，黄凉粉一般）。后乘地铁到春熙路，简单看了看熊猫屁股以及太古里到曲面大屏后前往宽窄巷子。宽窄巷子是成都少有的从清朝遗留下来的几条古街道，游览完宽窄巷子后乘地铁回酒店。 Day4：成都-猫鼻梁-四姑娘山双桥沟-日龙镇 接下来的2天，我们前往四姑娘山进行川西之旅。经过5个多小时的漫长车程，从海拔几百米到三千多米，我们到达了日隆镇。途中经过猫鼻梁观景台，我们在此简单休息并观景，在猫鼻梁处可以清楚地看到四姑娘山的全貌（大姑娘、二姑娘、三姑娘以及幺妹峰）。 在日隆镇吃过午饭后，便乘车前往双桥沟游览。进双桥沟后，先乘观光车到达最远点（原始森林），后沿着沟底一个景点一个景点进行游览并返回，不同景点间均有观光车进行接送。游览了大约4~5个小时后出沟，前往餐厅吃藏餐土火锅并参加篝火晚会。吃好后穿上藏服围着篝火简单跟着跳了一会后，自行步行返回酒店休息。 Day5：四姑娘山长坪沟-成都（建设巷、东郊记忆） 四姑娘山的第2天，我们前往长坪沟进行游览。乘观光车到达沟口后，全程徒步进行游览，有高山草甸、牦牛、溪流、瀑布、沼泽滩涂，以及若影若现的雪山。清晨云雾缭绕，到中午后天气逐渐清朗。长坪沟全程29公里，风景很美，但也很累。由于高原徒步及时间原因，此次只进沟约5公里后便原路返回，途中还爬了2次山坡前往高处观赏瀑布。月12点返回沟口，吃过午饭后，乘车返回成都（约5个小时，6点到达成都市区）。晚上去电子科大旁的建设巷美食街吃了很多小吃（撑死了）并简单逛了逛附近的东郊记忆，后返回酒店休息。 Day6：成都大熊猫繁育研究基地 国庆最后一天，我们一大早便前往熊猫基地进行游览。熊猫基地为成都的必玩景点，来成都一定要到基地来看看这些熊猫憨憨们。沿着大环线，吃竹子及睡觉的大熊猫千姿百态（坐着、躺着、挂着、趴着），月亮及太阳产房的幼年大熊猫们更是憨态可掬。其中还有幸看到了两只熊猫打架的场景，引得周围观众的阵阵高呼。景区了还有半开放饲养的小熊猫，以及放养的孔雀。天鹅湖中，还开着紫色的荷花，以及一黑一白两只天鹅在湖面上飘着。约中午1点游览结束离开基地，吃过午饭后简单逛了市区便前往双流机场附近的酒店休息，晚上还吃到了西昌火盆烧烤（味道还不错）。第二天一早便从双流机场返回合肥。
